Sobre mim

Desenvolvedor Back-End em formação, cursando Análise e Desenvolvimento de Sistemas pela UCSAL (conclusão prevista para 2027). Possuo conhecimentos no desenvolvimento de aplicações utilizando Go, Python e JavaScript/TypeScript, além de experiência com sistemas Linux (administração básica e linha de comando), modelagem e gerenciamento de bancos de dados PostgreSQL e versionamento com Git e GitHub. Com foco em arquitetura de sistemas e eficiência de software, tenho experiência prática no ciclo de implantação de aplicações, desde a conteinerização do Back-End até o deploy em ambientes de nuvem como Google Cloud Platform (GCP) e Supabase. Possuo domínio técnico na integração e consumo de APIs RESTful, garantindo a comunicação otimizada, performática e segura entre o cliente e o servidor através de ferramentas como Axios e React Query. Tenho forte interesse em Inteligência Artificial, focado em sua aplicação para a automação de processos e desenvolvimento de softwares mais inteligentes. Busco atuar na construção de soluções escaláveis, manutenção de sistemas e criação de novas funcionalidades, sempre aplicando boas práticas de programação, arquitetura de software e desenvolvimento colaborativo.

Oficina do Texto

Oficina do Texto é uma API RESTful em Go estruturada em camadas, com autenticação JWT e banco PostgreSQL (Supabase). A infraestrutura é containerizada com Docker e hospedada na nuvem via GCP (Cloud Run), utilizando a Vercel como proxy reverso para otimizar as requisições do cliente.
